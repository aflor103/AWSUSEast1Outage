<h2> Description </h2>
Analyzed the 2025 October AWS US-East-1 DNS-related outage and broke down single regional failure can have a drastic cascading affect through an entire cloud workload. In this project I focused on tracing the path of the DynamoDB failure and its rippling affect in the AWS cloud infrastructure. In the end, givign examples of how to mitigate these risk for future events.
